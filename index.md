## LLMSider

LLMSider is an AI copilot plugin for Obsidian with multi-model chat, in-editor writing help, agent tool use, MCP integration, semantic search, local skills, and optional local inference.

## Install

**Community Plugins**

Open Obsidian Settings -> Community Plugins -> Browse -> search for `LLMSider` -> Install -> Enable.

**BRAT**

1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat)
2. Click `Add Beta Plugin`
3. Enter `gnuhpc/obsidian-llmsider`
4. Enable the plugin

**Manual**

Download the latest release from [GitHub Releases](https://github.com/gnuhpc/obsidian-llmsider/releases), extract it to `YourVault/.obsidian/plugins/llmsider/`, reload Obsidian, and enable it.

## Start

1. Open `Settings -> LLMSider`
2. Add a connection and at least one model
3. Open `LLMSider: Open Chat`
4. Enable MCP, vector search, skills, or WebLLM if needed
