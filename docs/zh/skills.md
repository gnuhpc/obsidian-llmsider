# 🧩 Skills 与技能市场

技能（Skills）是 LLMSider 中的专项 AI 能力，它结合了特定指令、提示词和一组受限的工具，用于处理复杂任务。Skills 遵循与 [agentskills.io](https://agentskills.io) 兼容的标准格式。

---

## 🎯 什么是技能 (Skills)？

一个 **Skill** 是包含以下内容的独立文件夹：
- `SKILL.md` 或 `skill.json`：定义技能身份和权限的清单文件
- **指令 (Instructions)**：当此技能激活时，提供给 AI 的具体系统提示词
- **提示词模板 (Prompt Templates)**：技能内常用任务的快捷方式
- **工具白名单 (Tool Allowlist)**：此技能被授权使用的特定工具（内置或 MCP）

### 为什么要使用技能？

| 特性 | 普通聊天 | 技能 (Skills) |
|---------|---------------|--------|
| **系统身份** | 通用助手 | 专项专家（如：研究员、程序员） |
| **工具范围** | 所有启用的工具 | 仅与任务相关的工具 |
| **快捷方式** | 手动输入提示词 | 针对性的提示词模板 |
| **上下文感知** | 库级别 | 高度针对性的指令 |

## 🚀 技能市场 (Skills Market)

**技能市场** 是一个内置的画廊，您可以在其中发现并安装社区创建的技能。

### 如何使用市场：
1. 打开 **设置** → **LLMSider** → **技能市场 (Skills Market)**
2. 浏览或搜索技能（例如："网络调研"、"代码审查"、"每日回顾"）
3. 点击 **安装/下载** 将技能添加到您的本地库中
4. 安装后的技能将出现在 **本地技能** 列表和聊天界面中

---

## 📂 本地技能 (Local Skills)

您安装的技能存储在库中的指定文件夹中（默认：`skills/`）。

### 管理本地技能：
1. 前往 **设置** → **LLMSider** → **本地技能**
2. **启用/禁用**：单独切换技能的开启状态
3. **默认技能**：设置在开启新对话时自动激活的技能
4. **重新加载**：扫描技能目录以应用手动更改

---

## 🛠️ 创建自己的技能

您可以通过在 `skills/` 目录中创建一个包含 `SKILL.md` 文件的文件夹来构建自定义技能。

### SKILL.md 示例：

```markdown
---
name: 技术研究员
id: tech-researcher
description: 用于深度技术研究和文档生成的专业技能。
version: 1.0.0
allowed-tools: brave_web_search, read_url, write_file
metadata:
  preferred-mode: agent
---

你是一位资深技术研究员。你的目标是提供深入、准确且带有引用参考的技术分析。

在进行研究时：
1. 使用 Brave Search 查找最新信息。
2. 彻底阅读最相关的网页。
3. 将发现的结果综合成一份清晰的 Obsidian 笔记。
```

### 清单字段说明：
- `id`：唯一标识符（例如 `my-custom-skill`）
- `name`：UI 中显示的名称
- `description`：技能功能的简短摘要
- `allowed-tools`：以空格分隔的工具名称列表（例如 `list_dir read_file`）
- `instructions`：(可选) 系统指令。如果省略，则使用 YAML 之后的所有文本。
- `metadata.preferred-mode`：`normal` 或 `agent`。

## 💬 在聊天中使用技能

### 手动激活：
1. 在聊天界面，点击工具栏中的 **模式/技能选择器**。
2. 从列表中选择一个技能。
3. AI 现在将遵循该技能的指令，并且只能访问其允许的工具。

### 自动路由：
LLMSider 可以自动检测最适合您输入的内容的技能。
- 如果您问 "搜索网络关于 LLM 的趋势"，它可以自动激活 "技术研究员" 技能。
- 此行为由 **技能路由 (Skill Routing)** 设置控制。

### 技能提示词：
技能可以提供专门的提示词按钮。例如，"手账" 技能可能有 "晨间回顾" 或 "晚间复盘" 的按钮。

---

## 💡 最佳实践

1. **限制工具数量**：仅允许技能实际需要的工具，以提高 AI 的准确性和安全性。
2. **清晰的指令**：在 `SKILL.md` 正文中使用清晰、带有性格特征的指令。
3. **利用模板**：将常用操作添加为模板以节省输入时间。
4. **关注市场**：定期检查市场中您喜爱的社区技能的更新。
