# ✨ 选择弹出指南

## 概述

选择弹出提供对选定文本的即时 AI 驱动操作。只需在笔记中选择任何文本,右键单击,即可访问强大的 AI 转换,无需离开编辑器。

---

## 🎯 什么是选择弹出?

### 快速文本操作

使用 AI 在几秒钟内转换选定文本:

```
1. 选择文本
2. 右键单击(或使用快捷键)
3. 选择操作
4. 完成!
```

### 可用操作

| 操作 | 描述 | 示例 |
|--------|-------------|---------|
| **改进** | 增强清晰度和风格 | "好" → "优秀" |
| **简化** | 使文本更清晰 | 技术 → 简单 |
| **扩展** | 添加更多细节 | 简短 → 详细 |
| **总结** | 压缩内容 | 长 → 简短 |
| **修复语法** | 纠正错误 | 错误 → 正确 |
| **更改语气** | 调整正式程度 | 随意 → 专业 |
| **翻译** | 转换语言 | 英语 → 西班牙语 |
| **继续** | 生成更多 | 不完整 → 完整 |

## 🚀 入门

### 启用选择弹出

1. **打开设置**
   ```
   设置 → LLMSider → 选择弹出
   ```

2. **启用功能**
   ```yaml
   启用选择弹出: ✓
   右键单击显示: ✓
   快捷键: Cmd+Shift+A (可选)
   ```

3. **配置操作**
   - 选择要显示的操作
   - 自定义提示
   - 设置默认模型

### 基本用法

1. **选择文本**
   ```markdown
   [这段文本需要改进] ← 选择这个
   ```

2. **打开弹出**
   - 在选择上右键单击
   - 或按快捷键(Cmd+Shift+A)
   - 弹出出现在光标附近

3. **选择操作**
   ```
   ┌─────────────────────┐
   │ ✨ 改进写作         │
   │ 📝 简化            │
   │ 🔍 扩展            │
   │ 📊 总结            │
   │ ✓ 修复语法         │
   │ 🎨 更改语气        │
   │ 🌍 翻译            │
   │ ➡️ 继续            │
   └─────────────────────┘
   ```

4. **应用结果**
   - AI 处理选择
   - 显示内联差异
   - 接受(Tab)或拒绝(Esc)

## 💡 操作示例

### 1. 改进写作

**之前:**
```markdown
这个东西真的很好,可以很好地完成工作。
```

**之后:**
```markdown
这个工具表现出卓越的性能,并以出色的效率
有效地完成其预期目的。
```

### 2. 简化

**之前:**
```markdown
该实现利用复杂的算法方法
来促进计算资源的优化。
```

**之后:**
```markdown
代码使用智能方法使计算机运行更快。
```

### 3. 扩展

**之前:**
```markdown
AI 帮助写作。
```

**之后:**
```markdown
人工智能通过提供实时建议、改进语法和风格、
生成内容想法以及自动化重复任务来协助写作。
这项技术利用自然语言处理来理解上下文
并提供相关的高质量文本增强。
```

### 4. 总结

**之前:**
```markdown
机器学习是人工智能的一个子集,专注于使计算机能够从数据中学习
而无需显式编程。它使用统计技术为计算机系统提供
在特定任务上逐步改进性能的能力。该领域包括各种
方法,包括监督学习(模型从标记数据中学习)、
无监督学习(在未标记数据中寻找模式)以及
强化学习(代理通过试错学习)。
```

**之后:**
```markdown
机器学习使计算机能够使用统计方法从数据中学习。
关键方法包括监督、无监督和强化学习。
```

### 5. 修复语法

**之前:**
```markdown
他们要去商店买一些苹果和橙子。
```

**之后:**
```markdown
他们要去商店买一些苹果和橙子。
```

### 6. 更改语气

**专业 → 随意:**
```markdown
之前: 我写信通知您项目已完成。
之后: 嘿!只是想让你知道项目完成了!🎉
```

**随意 → 专业:**
```markdown
之前: 这个挺酷的,工作得还不错 imo
之后: 该解决方案展示了有效性并提供
      令人满意的结果。
```

### 7. 翻译

**英语 → 西班牙语:**
```markdown
之前: Hello, how are you today?
之后: Hola, ¿cómo estás hoy?
```

**英语 → 中文:**
```markdown
之前: Thank you for your help.
之后: 谢谢你的帮助。
```

## 💡 专业提示

### 🎯 有效的选择

**选择适当的量:**
```
✅ 选择完整的想法
✅ 包含周围上下文
❌ 不要选择部分单词
❌ 不要选择太多(>500 个单词)
```

**选择策略:**
```
单个单词: 点击单词
行: 三击
段落: Cmd+A(在段落中)
多个段落: 点击拖动
```
