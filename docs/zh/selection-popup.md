# 选择弹窗

## 概述

选择弹窗是你在编辑器、阅读视图或受支持网页内容里选中文本后出现的浮动操作栏。

它主要提供两个快速动作：

- 把选中文本加入 LLMSider 上下文
- 直接基于选中文本发起 Quick Chat

## 当前功能

当选中的文本达到一定长度时，LLMSider 会把操作按钮显示在选区附近，而不是要求你先打开完整聊天窗口。

当前代码里的行为包括：

- 支持编辑视图和阅读视图选区
- 阅读视图拖拽选中时，会避免误触点击
- 能识别受支持 webviewer 内容里的选区
- 开启自动引用时，可隐藏手动“添加到上下文”按钮
- 执行任务过程中会阻止继续添加上下文

## 相关设置

这个功能主要受两处设置影响：

- `Selection Popup`：是否显示选择弹窗动作
- `Context Settings -> Auto Reference`：是否自动把选中文本加入上下文

如果开启了自动引用，LLMSider 会优先自动更新当前选区上下文，而不是依赖手动点击“添加到上下文”。

## 常见用法

1. 在笔记中选中一段文字
2. 用弹窗把它加入上下文，或者直接发起 Quick Chat
3. 让 AI 改写、解释、翻译或继续编辑
4. 需要时应用差异结果
