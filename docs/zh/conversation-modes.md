# 对话模式

## 概述

LLMSider 现在有：

- `普通模式`
- `代理模式`
- 作为 `普通模式` 内部开关的 `超能力 (Superpower)`

## 普通模式

普通模式是默认对话流程。

它支持：

- 直接聊天
- 写作辅助
- 按需使用工具

即使内置工具、MCP 工具和 Skills 全部关闭，普通模式仍然可以作为纯聊天正常工作。

## 超能力 (Superpower)

超能力是叠加在普通模式之上的高级交互层，由专门的协调器 (Orchestrator) 管理。

它的特点包括：

- **交互式工作流**：提供分步追问和引导。
- **鲁棒性**：具备高级错误处理和自动重试机制。
- **选项渲染**：支持原生的 `CHOICE` 选项渲染供用户快速选择。
- **工具集成**：在保持用户知情的情况下无缝执行工具。
- **纯引导模式**：即使所有工具/技能都被禁用，也能作为结构化对话正常工作。

关键行为：

- 超能力在整个多步流程中始终专注于用户的最初目标。
- 相比于自主的“代理模式”，它提供了更接近人类直觉的交互体验。

## 代理模式

代理模式用于基于当前启用工具的自主多步骤执行。
