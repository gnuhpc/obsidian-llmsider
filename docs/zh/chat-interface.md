# 聊天界面

## 概述

聊天视图是 LLMSider 的主要工作区。

## 主要控件

顶部区域 (Header):

- **会话标题**：显示当前会话名称，点击即可进入编辑模式重命名。
- **清除聊天**：🗑️ 删除当前会话中的所有消息内容，但保留会话记录。
- **新建聊天**：➕ 开启一个全新的空白对话。
- **历史记录**：🕒 开启/关闭侧边栏历史会话列表，方便查找往期内容。
- **设置**：⚙️ 打开插件的详细配置页面。

输入区域与工具栏:

- **上下文附件**：📎 点击可将当前笔记、选中文本、整文件夹或网页内容加入 AI 的上下文。
- **对话模式**：在 **普通 (Normal)** 模式与 **代理 (Agent)** 模式之间切换。
- **超能力 (Superpower) 开关**：(S 图标) 开启分步式交互引导，由 AI 主动追问或分阶段完成复杂任务。
- **Skill 选择器**：✨ 选择已安装的技能，为当前对话应用特定的提示词逻辑或行为模式。
- **内置工具**：🔧 集中管理网页搜索、笔记操作、计算等 100+ 核心内置工具的调用权限。
- **MCP 插件控制**：🔌 管理 Model Context Protocol 服务，动态扩展 AI 的外部操作能力。
- **本地语义搜索**：🔍 开启/关闭基于向量数据库的笔记自动检索功能。
- **速读**：📖 为当前活动的笔记开启智能分析抽屉（摘要、思维导图、互动问答）。
- **优化提示词**：🪄 自动重写并增强你的输入内容，以获得更高质量的 AI 回复。
- **模型切换**：显示当前模型名称，点击可快速切换不同的 AI 提供商或各型号模型。
- **发送 / 停止**：⬆️ 发送输入内容，或在 AI 生成过程中点击 ⏹️ 强制停止。

消息操作 (Message Actions):

- **编辑**：✏️ 修改你已发送的消息，以此调整对话上下文。
- **复制**：📋 将消息内容复制到剪贴板。
- **重新生成**：🔄 让 AI 针对同一请求重新给出回复。
- **模型对比**：➕ 开启对比模式，在同一条消息下并排查看其他模型的回复。
- **应用 Diff**：🛠️（针对文件操作）预览并应用 AI 建议的代码或文本更改。

上下文建议 (Context Suggestions):

- **确认添加**：✓ 将自动推荐的相关文件加入当前对话上下文。
- **忽略建议**：✕ 关闭当前的文件推荐。

## 界面中的对话模式

当前 UI 暴露：

- `普通模式`
- 普通模式中的 `超能力` 开关
- `代理模式`

## 工具行为

在 `普通模式` 下，工具调用是可选的。

即使内置工具、MCP 工具和 Skills 都关闭，聊天仍然可以作为纯模型对话正常工作。

如果开启 `超能力`，即使没有可用工具，也应该继续渲染引导式选项，而不是退化成原始 `CHOICE` 文本。
